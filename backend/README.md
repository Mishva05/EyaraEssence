# Eyara Essence Backend

This directory is reserved for future backend development. Currently, all e-commerce functionality runs on the frontend with mocked API services and localStorage persistence.

## Future Scope
- User authentication and authorization (JWT/OAuth)
- Product database, inventory management, and search index
- Cart synchronization across devices
- Payment gateway integration (Stripe, Razorpay, etc.)
- Order processing, emails, and tracking
- Administrative dashboard
